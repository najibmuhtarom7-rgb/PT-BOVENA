# PT Bovena — Company Profile (Static Site)

Situs statis contoh untuk perusahaan peternakan, terinspirasi tata letak dan kejelasan konten dari situs korporat modern.

Cara menjalankan (Windows PowerShell):

- Buka `d:\pt_bovena` dan buka `index.html` langsung di browser.
- Atau jalankan server sederhana jika ingin melihat behavior CORS/paths:

```powershell
# jika ada Python 3
python -m http.server 8000; Start-Process http://localhost:8000
```

File penting:
- `index.html` — halaman utama
- `assets/css/style.css` — gaya
- `assets/js/main.js` — skrip kecil untuk navigasi mobile dan smooth scroll
- `assets/img/logo.svg`, `assets/img/farm.svg` — aset ilustrasi
